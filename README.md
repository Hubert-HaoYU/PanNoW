## Install with conda
1. download environment.yml to your target(your server or your personal computer)
2. run conda env create -f environment.yml
3. download PanNoW.py to to your target(your server or your personal computer)
4. conda activate PanNoW
5. python PanNoW.py

## Quick start
```bash
python PanNoW.py -i INPUT_DIR -x SUFFIX -o OUTPUT_DIR [-j THREADS] [-s SIMILARITY]
```
